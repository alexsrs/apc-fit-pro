  sistema de pontuação (ranking de decisão automática) para classificar cada usuário do APC Fit Pro em um dos 3 objetivos:
1. Controle de Doenças
2. Saúde e Bem-estar
3. Estética / Hipertrofia

⸻

Critérios de Classificação com Pontuação (Score System)

Cada pergunta-chave recebe um peso. A soma das respostas direciona o app a classificar o objetivo dominante.

⸻

PERGUNTAS-CHAVE E PESOS

Bloco 2 – Estado de Saúde e Histórico Médico
	•	Possui doença diagnosticada por médico?
	•	( ) Sim → +3 pontos para Controle de Doença
	•	( ) Não → +1 ponto para Saúde e Bem-estar
	•	Usa medicação contínua?
	•	( ) Sim → +2 pontos para Controle de Doença
	•	( ) Não → +1 ponto para Saúde e Bem-estar
	•	Já realizou cirurgia nos últimos 2 anos?
	•	( ) Sim → +1 ponto para Controle de Doença
	•	( ) Não → 0 pontos

⸻

Bloco 3 – Nível de Atividade
	•	Pratica atividade física atualmente?
	•	( ) Não → +2 pontos para Controle de Doença
	•	( ) Sim → +1 ponto para Saúde e Bem-estar
	•	Frequência semanal de treino:
	•	1-2x → +1 ponto Saúde e Bem-estar
	•	3-4x → +2 pontos Estética / Hipertrofia
	•	5x ou mais → +3 pontos Estética / Hipertrofia

⸻

Bloco 4 – Objetivo Declarado (Peso máximo)
	•	( ) Melhorar ou controlar uma doença crônica
→ +10 pontos Controle de Doença
	•	( ) Melhorar disposição, qualidade de vida e envelhecer com saúde
→ +10 pontos Saúde e Bem-estar
	•	( ) Reduzir gordura corporal, melhorar a estética ou ganhar massa muscular
→ +10 pontos Estética / Hipertrofia

⸻

Bloco 5 – Bem-estar
	•	Qualidade do sono (1 a 5):
	•	1-2 → +1 ponto Controle de Doença
	•	3-4 → +1 ponto Saúde e Bem-estar
	•	5 → +1 ponto Estética / Hipertrofia
	•	Nível de estresse:
	•	Alto → +1 ponto Controle de Doença
	•	Moderado → +1 ponto Saúde e Bem-estar
	•	Baixo → +1 ponto Estética / Hipertrofia

⸻

RANKING FINAL (Lógica de Decisão)
	•	Some os pontos acumulados para cada categoria.
	•	Categoria com maior pontuação = objetivo final.

Empates? Use o Objetivo Declarado (Bloco 4) como critério de desempate.
